## Description

### Specs
| Spec | Input | Output |
| :-------------     | :------------- | :------------- |
| **Program replaces "0" with "Beep!"** | 0 | Beep! |
| **Numbers that contain a "1" are replaced with "Boop!"** | 1 |  Boop! |
| **Numbers divisible by 3 are replaced with a phrase**| 9 | I'm sorry, Dave.  I'm afraid I can't do that
